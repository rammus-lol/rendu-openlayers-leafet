Bonjour voici le rendu de Paul Gréaume et Nicolas Herman. Vous pouvez cloner le repo puis après avoir: 
- Lancé le docker depuis \geonum_docker avec docker-compose up -d,
- Initialisé le geoserver dans le navigateur depuis [localhost:8080/geoserver](http://localhost:8080/geoserver) en entrant les informations de connections,
- Excuté depuis /geonum_docker/app les comandes npm run build et npm start.<br>

Le projet sera utilisable sur votre poste.
